# Hermes

Sistema de gestion comercial para pequenos negocios, orientado a una interfaz conversacional (ej. WhatsApp) con soporte para ventas, compras, inventario y fiados.

## Estado del proyecto

Este repositorio contiene documentacion inicial de arquitectura y lineamientos funcionales.

## Estructura

- `architecture.md`: arquitectura funcional y de datos del sistema.
- `docs/`: documentos de apoyo (plantillas y guias).

## Objetivo

Construir un MVP para operar un negocio pequeno desde chat, con foco en:

- Registro de compras y ventas.
- Control de inventario.
- Gestion de fiados y pagos.
- Reportes operativos basicos.

## Proximo paso sugerido

1. Definir stack tecnologico (backend, base de datos, integracion WhatsApp).
2. Diseñar API minima para modulo de ventas.
3. Implementar modelo de datos base y autenticacion por roles.
